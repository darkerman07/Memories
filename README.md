# Memories

## Introduction

**Memories** is a Full Stack MERN application built using **React, Node.js, Express, and MongoDB**.  
This simple social media app allows users to post and share interesting events from their lives.

By following this project, you'll gain a solid understanding of how the **MERN Stack** works from start to finish.

---

## Setup

To run the application locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/darkerman07/Memories.git
cd Memories
````

2. Install dependencies and start the app for both client and server:

```bash
# For server
cd server
npm install
npm start

# For client
cd client
npm install
npm start
```

3. Open your browser at `http://localhost:3000` to view the app.
